# Amazon Lookout for Metrics (amazon-lookout-for-metrics)
Amazon Lookout for Metrics uses machine learning to detect anomalies in your metrics data and help you determine their root causes. It allows you to monitor business and operational metrics such as revenue performance, customer engagement, and user activity.

### Amazon Lookout for Metrics API
The Amazon Lookout for Metrics API provides programmatic access to create and manage anomaly detectors, anomaly groups, alerts, and datasets for automated anomaly detection in business metrics.
